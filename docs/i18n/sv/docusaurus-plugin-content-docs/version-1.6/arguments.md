Du kan starta appen med argument.

`butterfly.exe <Path>`

Sökvägen är sökvägen till filen du vill öppna.

## Alternativ

|     Alternativ | Beskrivning                                         |
| --------------:|:--------------------------------------------------- |
| `--path`, `-p` | Välj datakatalog där all data från fjärilen lagras. |
