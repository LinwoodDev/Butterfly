Puteți porni aplicația cu argumente.

`butterfly.exe <Path>`

Calea către fișierul pe care doriți să îl deschideți.

## Opţiuni

|        Opţiune | Descriere                                                              |
| --------------:|:---------------------------------------------------------------------- |
| `--path`, `-p` | Alegeți dosarul de date unde sunt stocate toate datele de pe Butterfly |
