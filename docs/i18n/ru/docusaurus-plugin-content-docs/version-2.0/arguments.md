Вы можете запустить приложение с аргументами.

`butterfly.exe <Path>`

Путь к файлу, который вы хотите открыть.

## Варианты

|        Вариант | Описание                                                        |
| --------------:|:--------------------------------------------------------------- |
| `--path`, `-p` | Выберите каталог данных, в котором хранятся все данные Бабочки. |
