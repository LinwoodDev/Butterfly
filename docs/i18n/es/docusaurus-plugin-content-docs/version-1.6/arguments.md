Puede iniciar la aplicación con argumentos.

`butterfly.exe <Path>`

La ruta es la ruta al archivo que desea abrir.

## Opciones

|         Opción | Descripción                                                                   |
| --------------:|:----------------------------------------------------------------------------- |
| `--path`, `-p` | Elija el directorio de datos donde se almacenan todos los datos de Butterfly. |
