Con esta herramienta puede seleccionar elementos.

## Configuración

| Propiedad | Por defecto | Descripción                                           |
| ---------:|:-----------:|:----------------------------------------------------- |
|      modo | rectángulo  | Modo de selección. Válido son el rectángulo y el lazo |
