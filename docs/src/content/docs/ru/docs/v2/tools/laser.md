---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Через некоторое время рисунок исчезает.
Add a new drawing to the paper to cancel the previous laser stroke.

## Конфигурация

|            Свойства |     По умолчанию    | Описание                                                                                                              |
| ------------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|                Цвет |       Красный       | The color that will be drawn.                                                                         |
|       Ширина штриха |          5          | The width of the stroke.                                                                              |
|           Утончение | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|     Время видимости |          5          | The duration in seconds that the drawing will stay visible.                                           |
| Скрыть длительность | 0.5 | How long the laser stroke takes to disappear.                                                         |
|            Анимация |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

| Mode | Описание                                                            |
| ---: | :------------------------------------------------------------------ |
| Fade | The stroke fades out after the duration has passed. |
| Путь | The stroke disappears along the drawn path.         |
