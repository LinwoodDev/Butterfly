---
title: Barcode tool
---

The barcode tool allows you to add machine-readable codes to the canvas.
You can use it to create QR codes, Data Matrix codes, and Code 128 barcodes.

## Usage

1. Select the barcode tool.
2. Choose the barcode type in the tool configuration.
3. Enter or paste the content that should be encoded.
4. Click or drag on the canvas to place the barcode.

The generated barcode is added as an element on the canvas. You can move, resize, arrange, or delete it like other elements.

## Barcode types

|           ପ୍ରକାର | ବିବରଣୀ                                                                                                 |
| ---------------: | :----------------------------------------------------------------------------------------------------- |
|          QR କୋଡ୍ | A square 2D code that is commonly used for links, short text, and contact information. |
| ଡାଟା ମ୍ୟାଟ୍ରିକ୍ସ | A compact 2D code that is useful when the barcode should stay small.                   |
|         କୋଡ୍ 128 | A 1D barcode for alphanumeric data.                                                    |

## Configuration

|        Property | Default | ବିବରଣୀ                                                    |
| --------------: | :-----: | :-------------------------------------------------------- |
| ବାର୍କୋଡ୍ ପ୍ରକାର | QR କୋଡ୍ | The type of barcode that will be created. |
|            ରଙ୍ଗ |  Black  | The color of the generated barcode.       |
