---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Через деякий час малюнок згасає.
Add a new drawing to the paper to cancel the previous laser stroke.

## Конфігурація

|          Властивість |        Типово       | Опис                                                                                                                  |
| -------------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|                Колір |       Червоний      | The color that will be drawn.                                                                         |
|        Ширина штриха |          5          | The width of the stroke.                                                                              |
|                Думки | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|           Тривалість |          5          | The duration in seconds that the drawing will stay visible.                                           |
| Приховати тривалість | 0.5 | How long the laser stroke takes to disappear.                                                         |
|             Анімація |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

| Mode | Опис                                                                |
| ---: | :------------------------------------------------------------------ |
| Fade | The stroke fades out after the duration has passed. |
| Шлях | The stroke disappears along the drawn path.         |
