## Configuração

|            Propriedade | Padrão | Descrição                                                                            |
| ---------------------: | :----: | :----------------------------------------------------------------------------------- |
|     Espessura do traço |    5   | A largura do traço                                                                   |
| Multiplicador de traço |    1   | Se você tem uma caneta digital, este é o multiplicador que será adicionado à pressão |
