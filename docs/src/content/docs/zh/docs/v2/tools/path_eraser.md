---
title: Path eraser tool
---

With this tool you can delete the whole path on the paper.

## 配置

|    属性 |  默认设置 | 描述                                         |
| ----: | :---: | :----------------------------------------- |
|  笔画宽度 |   5   | 笔画的宽度                                      |
|  笔记倍数 |   1   | 如果您有一个样式表，这是将添加到压力中的乘数                     |
| 包括擦除器 | false | 如果您可以删除擦除器的路径，此设置将被控制。 删除的擦除器下的路径将会显示 笔记倍数 |
