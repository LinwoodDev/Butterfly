---
title: Stamp tool
---

The stamp tool allows you to add components to your canvas.
使用现有的包，创建您自己或导入一个包来使用它。 Read more [here](../../pack).

## 配置

| 属性 | 默认设置 | 描述                         |
| -: | :--: | :------------------------- |
|  包 |   无  | 组件所在的包                     |
| 组件 |   无  | 将添加到画布的组件. |
