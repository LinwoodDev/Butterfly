---
title: Pen tool
---

With this tool you can draw on this paper.

## 配置

|   财产 |  默认设置 | 描述                     |
| ---: | :---: | :--------------------- |
|   颜色 |   黑色  | 将绘制的颜色                 |
| 笔画宽度 |   5   | 笔画的宽度                  |
| 笔记倍数 |   1   | 如果您有一个样式表，这是将添加到压力中的乘数 |
| 依靠缩放 | false | 这将改变基于缩放级别的笔画宽度。       |
