---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Hetken kuluttua piirros katoaa pois.
Add a new drawing to the paper to cancel the previous laser stroke.

## Asetukset

|     Ominaisuus |        Oletus       | Kuvaus                                                                                                                |
| -------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|           Väri |       Punainen      | The color that will be drawn.                                                                         |
| Aivojen leveys |          5          | The width of the stroke.                                                                              |
|     Hehkuminen | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|          Kesto |          5          | The duration in seconds that the drawing will stay visible.                                           |
|  Piilota kesto | 0.5 | How long the laser stroke takes to disappear.                                                         |
|      Animaatio |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

|  Mode | Kuvaus                                                              |
| ----: | :------------------------------------------------------------------ |
|  Fade | The stroke fades out after the duration has passed. |
| Polku | The stroke disappears along the drawn path.         |
