---
title: "Ferramentas"
---

Tools can edit the paper.

Use a barra lateral para aprender mais sobre as ferramentas.

Create a new tool by clicking on the plus icon. Change the tool order by dragging and change the tool by clicking.
Se você clicar em uma ferramenta selecionada, você pode configurá-la.

The name of the tool will be displayed as tooltip on hover on the tool.

## Temporary tools

There can be only one temporary tool at a time. It appears on the left of the toolbar and will disappear when another tool is selected.
