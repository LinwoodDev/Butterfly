:::note

Esta es una aplicación web progresiva. Puedes instalarlo en tu navegador y usarlo sin conexión.
Todos los datos se almacenan en su navegador.

The web version works on all modern browsers, but is more performant on Chromium-based browsers due to their support for WebAssembly (Wasm). Firefox and Safari currently do not support the specific Wasm features required. See [Flutter's browser compatibility documentation](https://docs.flutter.dev/platform-integration/web/wasm#learn-more-about-browser-compatibility) for more details.

:::

## Versiones

Las versiones principales y la vista previa de Butterfly no son las mismas que las versiones estables y nocturnas.
No están directamente asociados con una liberación. Se actualizan tan pronto como un nuevo commit se inserta en el repositorio.
