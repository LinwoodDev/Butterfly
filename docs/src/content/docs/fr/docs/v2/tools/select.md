---
title: Outil sélection
---

Avec cet outil vous pouvez sélectionner des éléments.

## Configuration

| Propriété | Par défaut | Libellé                                                                                                |
| --------: | :--------: | :----------------------------------------------------------------------------------------------------- |
|      Mode |  Rectangle | Mode de sélection. Les éléments valides sont le rectangle et le lasso. |
