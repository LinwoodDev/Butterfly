## Le programme bêta sur le Play Store

Click on [this link](https://play.google.com/apps/testing/dev.linwood.butterfly) to register for the [nightly](/community/nightly) version of the app. Veuillez noter que cette version peut avoir des bugs. La version Play Store sera automatiquement mise à jour lorsqu'une nouvelle version nocturne sera publiée.
