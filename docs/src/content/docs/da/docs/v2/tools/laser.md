---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Efter et stykke tid forsvinder tegningen.
Add a new drawing to the paper to cancel the previous laser stroke.

## Konfiguration

|       Egenskab |       Standard      | Varebeskrivelse                                                                                                       |
| -------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|          Farve |         Rød         | The color that will be drawn.                                                                         |
|     Slaglængde |          5          | The width of the stroke.                                                                              |
|      Udtynding | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|       Varighed |          5          | The duration in seconds that the drawing will stay visible.                                           |
| Skjul varighed | 0.5 | How long the laser stroke takes to disappear.                                                         |
|      Animation |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

| Mode | Varebeskrivelse                                                     |
| ---: | :------------------------------------------------------------------ |
| Fade | The stroke fades out after the duration has passed. |
|  Sti | The stroke disappears along the drawn path.         |
