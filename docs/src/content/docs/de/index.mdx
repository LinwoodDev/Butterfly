## Prinzipien

<CardGrid stagger>
  <Card title="Plattformübergreifend" icon="laptop">
    Sie können die App auf Ihrem Smartphone, Tablet, Laptop oder Desktop verwenden. Die App ist für Linux, Android und Windows verfügbar. Sie können die App auch im Web verwenden.
  </Card>
  <Card title="Einfach und intuitiv" icon="puzzle">
    Jedes Werkzeug ist am richtigen Platz. Öffnen Sie die App und starte. Ändern Sie Ihre Werkzeuge, indem Sie darauf klicken.
  </Card>
  <Card title="Anpassbar" icon="pencil">
    Passen Sie alles an Ihre Bedürfnisse an. Wählen Sie Ihre eigene Farbe, erstellen Sie eine Palette und fügen Sie Ihre Seiten dem Papier hinzu. Das Papier hat eine unendliche Größe, perfekt für Ihre Ideen und Noten.
  </Card>
  <Card title="Wählen Sie aus, wo Ihre Daten gespeichert sind" icon="open-book">
    Sie können wählen, ob Ihre Daten lokal oder in Ihrer bevorzugten Cloud (WebDAV) gespeichert werden. Alternativ können Sie Ihre Daten in eine Datei exportieren und später wieder importieren.
  </Card>
</CardGrid>
