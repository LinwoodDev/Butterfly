## Minimale Systemvoraussetzungen

- Android Nougat 7.0 (API-Level 24) oder höher.

:::tip[What apk soll ich verwenden?]

**Universal** sollte auf allen Geräten funktionieren. Wenn Sie eine kleinere APK möchten, verwenden Sie die APK, die mit der Architektur Ihres Geräts übereinstimmt.
**Legacy**-APKs verwenden beim Build das Flag `useLegacyPackaging`, das die nativen Bibliotheken komprimiert, um die APK-Größe zu reduzieren. Dies kann jedoch die Ladeleistung beeinflussen. Lies mehr [in der Android-Dokumentation](https://developer.android.com/reference/tools/gradle-api/7.1/com/android/build/api/dsl/JniLibsPackagingOptions#uselegacypackaging).

:::
