## Windows SmartScreen

Windows warnt Benutzer, wenn Software ohne Zertifikat installiert wird.

![SmartScreen](@assets/img/smart-screen.png)

Um die App zu installieren, müssen Sie auf „Weitere Informationen“ klicken.

![SmartScreen weitere Informationen](@assets/img/smart-screen-more-info.png)

Klicken Sie anschließend auf „Trotzdem ausführen“.
