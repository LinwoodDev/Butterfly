## Versionen

Die Haupt- und die Preview-Versionen von Butterfly sind nicht die gleichen wie die Stable-und Night-Versionen.
Sie sind nicht direkt mit einer Veröffentlichung verbunden. Sie werden aktualisiert, sobald ein neuer Commit ins Projektarchiv verschoben wird.
