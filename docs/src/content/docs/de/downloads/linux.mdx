## Binärdateien

Bitte installiere `libsecret-1-dev` und `libjsoncpp-dev`, bevor Sie Butterfly installierst.

Die alternativen Binärdateien wurden mit Ubuntu 22.04 LTS statt Ubuntu 24.04 LTS gebaut und sollten auf Geräten mit einer älteren glibc-Version funktionieren.
