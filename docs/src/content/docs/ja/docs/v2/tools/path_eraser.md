---
title: Path eraser tool
---

With this tool you can delete the whole path on the paper.

## 設定

|       属性 | デフォルト | 説明                                                   |
| -------: | :---: | :--------------------------------------------------- |
|  ストロークの幅 |   5   | ストロークの幅                                              |
|  ストローク乗数 |   1   | スタイラスを持っている場合、これは加算される乗数です                           |
| 消しゴムを含める | false | この設定は、消しゴムのパスを削除できるかどうかを制御します。 削除された消しゴムの下のパスが表示されます |
