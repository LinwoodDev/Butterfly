---
title: Laser tool
---

With this tool you can draw on this paper temporarily. وبعد فترة من الزمن، يتلاشى الرسم.
Add a new drawing to the paper to cancel the previous laser stroke.

## الإعدادات

|   الممتلكات |      الافتراضي      | الوصف                                                                                                                 |
| ----------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|       اللون |         أحمر        | The color that will be drawn.                                                                         |
|   سُمك الخط |          5          | The width of the stroke.                                                                              |
|     الترقيق | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|       المدة |          5          | The duration in seconds that the drawing will stay visible.                                           |
| مدة الإخفاء | 0.5 | How long the laser stroke takes to disappear.                                                         |
|      الحركة |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

|   Mode | الوصف                                                               |
| -----: | :------------------------------------------------------------------ |
|   Fade | The stroke fades out after the duration has passed. |
| المسار | The stroke disappears along the drawn path.         |
