### الشبكة

الشبكة تسمح لك بعناصر الموقع بالضبط. يمكنك استخدامه لنقل العناصر أو إنشاء الأشكال.

### حاكم

يسمح لك الحاكم بقياس المسافة بين نقطتين. بالإضافة إلى ذلك يمكنك استخدامه لإنشاء خط مع أداة القلم.
