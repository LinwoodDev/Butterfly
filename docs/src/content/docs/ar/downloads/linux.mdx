## ثنائيات

Please install `libsecret-1-dev` and `libjsoncpp-dev` before installing Butterfly.

The alternative binaries were built using Ubuntu 22.04 LTS (instead of Ubuntu 24.04 LTS) and should work on devices running an older version of glibc.
