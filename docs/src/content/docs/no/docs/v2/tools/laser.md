---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Etter en stund unngår tegningen seg vekk.
Add a new drawing to the paper to cancel the previous laser stroke.

## Konfigurasjon

|             Egenskap |       Standard      | Beskrivelse                                                                                                           |
| -------------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|                Farge |         Rød         | The color that will be drawn.                                                                         |
| Bredde på slagbredde |          5          | The width of the stroke.                                                                              |
|                 Tynn | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|             Varighet |          5          | The duration in seconds that the drawing will stay visible.                                           |
|       Skjul varighet | 0.5 | How long the laser stroke takes to disappear.                                                         |
|            Animasjon |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

| Mode | Beskrivelse                                                         |
| ---: | :------------------------------------------------------------------ |
| Fade | The stroke fades out after the duration has passed. |
|  Sti | The stroke disappears along the drawn path.         |
