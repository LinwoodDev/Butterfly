---
title: Argumenty
---

Aplikaci můžete spustit pomocí argumentů.

`butterfly.exe <Path>`

Cesta je cesta k souboru, který chcete otevřít.

## Možnosti

|        Možnost | Popis                                                           |
| -------------: | :-------------------------------------------------------------- |
| `--path`, `-p` | Vyberte datový adresář, kde jsou uložena všechna data Butterfly |
