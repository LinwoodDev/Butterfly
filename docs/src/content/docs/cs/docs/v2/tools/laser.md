---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Po nějaké době se rýsuje pryč.
Add a new drawing to the paper to cancel the previous laser stroke.

## Konfigurace

|            Majetek |       Výchozí       | Popis                                                                                                                 |
| -----------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|              Barva |       Červená       | The color that will be drawn.                                                                         |
|              Šířka |          5          | The width of the stroke.                                                                              |
|         Ztenčování | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|        Doba trvání |          5          | The duration in seconds that the drawing will stay visible.                                           |
| Doba trvání skrytí | 0.5 | How long the laser stroke takes to disappear.                                                         |
|            Animace |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

|  Mode | Popis                                                               |
| ----: | :------------------------------------------------------------------ |
|  Fade | The stroke fades out after the duration has passed. |
| Cesta | The stroke disappears along the drawn path.         |
