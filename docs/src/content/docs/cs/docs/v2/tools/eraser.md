title: Eraser tool
---

With this tool you can erase the elements on the paper.

## Konfigurace

|            Majetek | Výchozí | Popis                                                          |
| -----------------: | :-----: | :------------------------------------------------------------- |
|              Šířka |    5    | Šířka zdvihu                                                   |
| Multiplikátor tahů |    1    | Máte-li stylus, je to multiplikátor, který bude přidán k tlaku |
