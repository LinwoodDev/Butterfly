---
title: Laser tool
---

With this tool you can draw on this paper temporarily. After a while, the drawing fades away.
Add a new drawing to the paper to cancel the previous laser stroke.

## Configuration

|     Property |       Default       | Mô tả                                                                                                                 |
| -----------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|      Màu sắc |         Red         | The color that will be drawn.                                                                         |
|   Độ dày nét |          5          | The width of the stroke.                                                                              |
|     Làm mảnh | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|   Thời lượng |          5          | The duration in seconds that the drawing will stay visible.                                           |
| Thời gian ẩn | 0.5 | How long the laser stroke takes to disappear.                                                         |
|     Hoạt ảnh |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

|      Mode | Mô tả                                                               |
| --------: | :------------------------------------------------------------------ |
|      Fade | The stroke fades out after the duration has passed. |
| Đường dẫn | The stroke disappears along the drawn path.         |
