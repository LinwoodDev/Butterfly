---
title: Laser tool
---

With this tool you can draw on this paper temporarily. Efter ett tag bleknar teckningen bort.
Add a new drawing to the paper to cancel the previous laser stroke.

## Konfiguration

|         Egenskap |       Standard      | Beskrivning                                                                                                           |
| ---------------: | :-----------------: | :-------------------------------------------------------------------------------------------------------------------- |
|             Färg |         Röd         | The color that will be drawn.                                                                         |
|     Stroke bredd |          5          | The width of the stroke.                                                                              |
|         Gallring | 0.4 | The effect of pressure on the stroke size. Set it to `0` for a constant stroke width. |
|      Varaktighet |          5          | The duration in seconds that the drawing will stay visible.                                           |
| Dölj varaktighet | 0.5 | How long the laser stroke takes to disappear.                                                         |
|        Animering |         Fade        | The animation used when the laser stroke disappears.                                                  |

## Animation modes

|   Mode | Beskrivning                                                         |
| -----: | :------------------------------------------------------------------ |
|   Fade | The stroke fades out after the duration has passed. |
| Sökväg | The stroke disappears along the drawn path.         |
